# 开源合规声明

## 项目信息

- 项目名称：知喻 - 多智能体协同课后学习平台
- 项目用途：广州大学第二届"庆园杯"人工智能创新应用大赛 主赛道主题三参赛作品

## 使用到的开源项目

### 无直接依赖的开源项目
本项目不使用 SillyTavern、MinerU、bge-m3、bge-reranker-v2 等开源项目作为代码依赖。
所有功能均基于以下通用开源库自主开发：

| 库名 | 许可证 | 用途 |
|------|--------|------|
| FastAPI | MIT License | Web 框架 |
| SQLAlchemy | MIT License | ORM |
| ChromaDB | Apache 2.0 | 向量数据库 |
| Bootstrap | MIT License | 前端 UI |
| PyMuPDF | AGPL-3.0 (GPL兼容) | PDF 文档解析 |
